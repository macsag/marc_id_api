KHW BN API

API wzbogacające rekordy bibliograficzne BN o numery rekordów wzorcowych KHW w podpolu |0

Dostępne metody

/get_bibs/{zapytanie do data.bn.org.pl}

Metoda zwraca żądane rekordy bibliograficzne wzbogacone o identyfikatory rekordów wzorcowych; w zapytaniu należy pominąć prefix rodzaju rekordu i formatu, czyli "bibs.xml?".

Przykładowe poprawne zapytanie: /get_bibs/createdDate=2018-11-13T10%3A00%3A00Z%2C2018-11-13T11%3A00%3A00Z&limit=100
